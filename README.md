
#  Application Android – Gestion des Étudiants (SQLite)

##  Objectif

Ce projet consiste à développer une application Android permettant de gérer des étudiants en utilisant une base de données locale SQLite.

L’application implémente les opérations CRUD :

* ➕ Ajouter un étudiant
* 🔍 Rechercher un étudiant par ID
* ❌ Supprimer un étudiant
